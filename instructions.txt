Hey! Here is everything you need to know to get the GestureVoice AI project running on your machine. 

This project is split into two parts: a Python backend (which handles the AI translation and the voice generation) and a React frontend (which handles the webcam and the UI).

### Prerequisites
Before you start, make sure you have installed on your laptop:
1. Python (version 3.9 or higher)
2. Node.js (download it from their official website if you don't have it)

### Step 1: Set up the Python Backend
The backend uses FastAPI, Google's Gemini API, and Microsoft Edge TTS.

1. Open a terminal and navigate to the `backend` folder.
2. Install the required Python libraries by running:
   pip install -r requirements.txt
3. Start the Python server:
   python -m uvicorn app:app --reload

(Keep this terminal open, it needs to stay running!)

### Step 2: Set up the React Frontend
The frontend uses Vite, React, and Google's MediaPipe for ultra-fast hand tracking.

1. Open a second, new terminal window and navigate to the `frontend` folder.
2. Install the necessary node packages:
   npm install
3. Start the website:
   npm run dev

### Step 3: Use the App!
1. Open your web browser and go to `http://localhost:5173`.
2. Allow the browser to access your webcam.
3. Show your hand to the camera to build a sentence using the gestures on the right side of the screen.
4. Press SPACEBAR to have the AI translate your gestures into a natural English sentence and speak it out loud!
5. Press 'C' to clear your current words.

Note: Since the backend uses the Gemini AI to build the sentences, you might be prompted to enter a Gemini API key the first time you run it. Just paste it in the terminal if it asks!

Enjoy!
